# üìù Blogging App

A full-stack **MERN (MongoDB, Express, React, Node.js)** blogging application that allows users to **create, view, update, and delete blogs**. User authentication ensures only logged-in users can manage their own blogs. The app is fully responsive and visually displays blogs using **cards**.

üîó **Live Demo:** [https://blogging-app-lovat.vercel.app/](https://blogging-app-lovat.vercel.app/)

---

## Features

### Authentication
- **Signup & Login** system
- Secure session-based authentication
- Only authenticated users can create, edit, or delete their blogs

### Blog Management
- **Create Blog:** Add a title, body, and image
- **My Blogs:** View, edit, or delete your own blogs
- **All Blogs:** View all blogs from all users

### UI & UX
- Responsive design using **Bootstrap**
- Blogs displayed as **cards** with images, title, and date
- Interactive buttons for edit and delete actions
